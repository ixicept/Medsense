# MedSense

**AOL Softeng - Kelompok 6**  
*A healthcare application for medical analysis*

## Getting Started

### Prerequisites
- Node.js (for Frontend)
- Go (for Backend)
- npm/yarn

### Installation & Setup

#### Frontend
```bash
cd ./FrontEnd\ -\ Medsense/
npm install       # Install dependencies
npm run dev      # Start development server
```

#### Backend
```bash
cd ./BackEnd\ -\ Medsense/
go run main.go
```
